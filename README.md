# spotify

This project contains a data analysis notebook focused on exploring Spotify music trends for 2024.

Overview:
- Data cleaning
- Exploratory Data Analysis
- Cross-platform popularity comparison
- Outlier detection on streaming metrics
- Machine Learning:
    - KMeans clustering on audio & popularity features
    - Regression models predicting Spotify streams (Baseline Linear Model + Random Forest)

Technologies used:
- Python 3
- Jupyter Notebook
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (KMeans, RandomForestRegressor)

How to run: 
git clone https://github.com/<your-repo-url>.git
cd <project-folder>
pip install -r requirements.txt
jupyter notebook spotify_2024.ipynb
